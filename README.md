#app name
Color Generator
#About
It is a basic React app color generator that generate color values that utilized the Values.js library
#How to use
Simply paste your desired color code value and click the generate button to generate various weight values of the same code. Hover on around any of the color code valse
to pop out the copy text button. Click to button to copy the code values.
